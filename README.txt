# PUSTAK

•	This application provides effective platform for buying and selling of second hand books.
•	It gives the information about the availability of desired new book in the nearest bookstore from the current location.
•	It allows the bookstore owners to register their bookstore and lets them manage their store.
•	The users can view their data from any device anywhere, as data is stored in cloud.
•	Bookstores can advertise themselves by registering with this application.
•	The users can view the availability of books at different bookstores, starting with the one closest to them.
